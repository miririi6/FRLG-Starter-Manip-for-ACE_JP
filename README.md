# 概要
FRLG任意コード実行RTAにおいて、バグポケモンの種となりつつ高個体値な御三家を検索するためのソフトウェアです。  
個体値、性格、必要経験値等でフィルタをかけて検索できます。  
バグポケモンの作成方法には経験値をつかうものと、努力値を使うものがありますが、現状このソフトでは経験値にのみ対応しています。  
# 使い方
## 検索方法
以下の項目を設定し、検索ボタン（または数値入力ボックス上でのエンターキー）で条件に合うポケモンを検索します。
### 要求個体値
入力数値以上の個体値のポケモンのみがフィルタを通過します。  
左から、下降補正・補正無し・上昇補正時の値です。
### 性格
チェックをつけた性格のポケモンのみがフィルタを通過します。
### 最小・最大経験値
指定した範囲内の経験値でバグポケモン種となれるポケモンのみがフィルタを通過します。  
最小経験値+87未満かつ奇数ズレの場合、育て屋での調整ができないためフィルタで弾いています。
### 最小・最大フレーム
指定範囲内のポケモンを検索します。
### トレーナーID
検索のシード値です。  
ゲーム開始後、トレーナーカードを見て入力します。
